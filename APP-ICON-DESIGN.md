# InboxIQ - App Icon Design Concepts

**Date:** 2026-03-04
**Brand:** ClearPointLogic / InboxIQ
**Theme:** Clean, modern, professional with AI magic
**Platform:** iOS (App Store + Device)

---

## 🎨 Design Requirements (Apple)

### Technical Specs
- **Size:** 1024×1024 pixels (App Store)
- **Format:** PNG (no transparency)
- **Color Space:** sRGB or Display P3
- **Shape:** Square (iOS adds rounded corners automatically)
- **Safe Area:** Keep important elements 10% from edges

### Design Principles
- **Simple:** Recognizable at small sizes (40×40)
- **Unique:** Stands out in App Store search
- **Memorable:** Users remember it
- **Relevant:** Communicates email + AI

## 🛠️ How to Create (3 Options)

### Option A: Canva (Easiest, Free)
1. Go to Canva.com
2. Create custom size: 1024×1024
3. Search templates: "app icon"
4. Use shapes:
   - Envelope: Rectangle + triangle flap
   - Gradient: Background > Gradient tool
   - Sparkles: Star shapes, gold color
5. Export as PNG (no background)

**Time:** 30 minutes
**Cost:** Free
**Skill:** Beginner

### Option B: Figma (Professional)
1. Create 1024×1024 frame
2. Draw envelope with pen tool
3. Apply linear gradient
4. Add sparkle vector icons
5. Export as PNG @1x

**Time:** 1 hour
**Cost:** Free
**Skill:** Intermediate

After creating 1024×1024, generate these sizes:

|--------|-----------|----------|----------|

| Spotlight | 40×40 | 80×80 | 120×120 |
| Settings | 29×29 | 58×58 | 87×87 |

**Xcode generates these automatically** from your 1024×1024 master.

## 💭 Design Tips

1. **Test at 40×40** - If it's unrecognizable, simplify
2. **High contrast** - Must work on light AND dark backgrounds
3. **No text** - "InboxIQ" won't be readable at small sizes
4. **Bold shapes** - Thin lines disappear at small sizes
5. **Limited colors** - 2-3 colors max for clarity
6. **Unique angle** - Don't copy Gmail/Outlook exactly

## 🔍 Competitor Analysis (What NOT to do)

**Gmail:** Red M shape - too simple, generic
**Outlook:** Blue O with envelope - corporate, dated
**Spark:** Blue lightning - energetic but cluttered
**Superhuman:** Pink/Blue gradient H - too abstract

**InboxIQ should be:**
- Cleaner than Spark
- More modern than Outlook
- More AI-focused than Gmail
- More professional than Superhuman
