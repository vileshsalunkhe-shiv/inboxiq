# InboxIQ App Icon - Canva Step-by-Step Tutorial

**Design:** Concept 1 "AI Envelope"
**Time:** 30 minutes
**Difficulty:** Beginner-friendly
**Result:** 1024×1024 PNG app icon

---

## 🎨 What We're Creating

A modern envelope icon with:
- Blue → Purple gradient background
- Clean white envelope
- Gold sparkles (AI magic)
- Professional, minimal, memorable

---

## 📋 Step-by-Step Instructions

### Step 1: Set Up Canvas (2 minutes)

1. Go to **Canva.com** (create free account if needed)
2. Click **"Create a design"** (top right)
3. Click **"Custom size"**
4. Enter: **1024 × 1024** pixels
5. Click **"Create new design"**

You should now see a blank white square canvas.

---

### Step 2: Add Gradient Background (5 minutes)

1. Click the **Background** button (left sidebar)
2. Click **"Gradient"** tab
3. Click **"Linear gradient"** (or any gradient)
4. Click the gradient to edit colors

**Set gradient colors:**
- **First color (top-left):** `#007AFF` (iOS blue)
  - Click first color circle
  - Paste `007AFF` in hex field
  - Or use color picker: bright blue
  
- **Second color (bottom-right):** `#5856D6` (purple)
  - Click second color circle
  - Paste `5856D6` in hex field
  - Or use color picker: medium purple

**Adjust gradient angle:**
- Rotate the gradient line to go from **top-left to bottom-right** (diagonal, ~135°)
- Looks like this: `/` direction

Your background should now be a smooth blue-to-purple gradient.

---

### Step 3: Create Envelope Base (10 minutes)

**Add rectangle (envelope body):**
1. Click **"Elements"** (left sidebar)
2. Search: `rectangle`
3. Drag a **rectangle** to canvas
4. Resize to approximately: **600 × 450** pixels (use top toolbar size fields)
5. Center it on canvas (drag or use position tools)

**Style the rectangle:**
1. Click rectangle
2. Set **color** to **white** (`#FFFFFF`)
3. Click **"Effects"** (top toolbar)
4. Add **"Shadow"** effect
   - Blur: 20
   - Transparency: 50%
   - Angle: 135° (bottom-right)
5. Set **corner radius** to **16** (rounded corners, top toolbar)

**Add triangle (envelope flap):**
1. Click **"Elements"** → **"Shapes"**
2. Search: `triangle`
3. Drag a **triangle** to canvas
4. Resize to approximately: **600 × 200** pixels (wide, short triangle)
5. Position it **overlapping top of rectangle** (looks like envelope flap)
6. Rotate triangle so point is **facing down** (flip if needed)
7. Set color to **white** (`#FFFFFF`)
8. Add same **shadow** effect

**Fine-tune alignment:**
- Select both shapes (click + Shift-click)
- Use alignment tools (top toolbar) to **center horizontally**
- Envelope should look like: □ with △ on top

---

### Step 4: Add Sparkles (10 minutes)

1. Click **"Elements"** (left sidebar)
2. Search: `sparkle` or `star` or `glitter`
3. Find **star/sparkle icons** (preferably simple, not photos)

**Add 6-8 sparkles around envelope:**
1. Drag a sparkle to canvas
2. Resize to **small** (32-48 pixels)
3. Set color to **gold/yellow** (`#FFD60A`)
4. Click **"Effects"** → Add **"Glow"**
   - Color: Same gold
   - Size: 8-12
   - Transparency: 60%
5. Duplicate sparkle (Ctrl+D or Cmd+D)
6. Place sparkles around envelope:
   - Top-left corner (2-3 sparkles)
   - Top-right corner (2-3 sparkles)
   - Bottom-left (1-2 sparkles)
   - Bottom-right (1-2 sparkles)
7. **Vary sizes:** Some larger (48px), some smaller (24px)
8. **Rotate randomly:** Give them different angles for dynamic feel

**Pro tip:** Use different sparkle icons for variety (4-point stars, 6-point stars, simple stars)

---

### Step 5: Optional: Add Lightning Bolt (5 minutes)

If you want extra "AI power" vibe:

1. Search **"lightning bolt"** in Elements
2. Find a **simple, thin lightning icon**
3. Resize to **64-80 pixels**
4. Place near **top-right of envelope**
5. Color: **Gold** (`#FFD60A`)
6. Add **glow** effect (same as sparkles)

---

### Step 6: Final Touches (5 minutes)

**Adjust layering:**
1. Right-click envelope shapes → **"Bring to front"** (envelope on top)
2. Right-click sparkles → **"Send to back"** (behind envelope, but visible)

**Check composition:**
- Envelope should be **centered**
- Sparkles should **frame the envelope** nicely
- Nothing too close to edges (leave ~100px margin)
- Gradient should be visible around envelope

**Test at small size:**
1. Zoom out (Ctrl+- or pinch out)
2. Check if icon is recognizable when small
3. If sparkles are too small/busy, remove 1-2

---

### Step 7: Export (3 minutes)

1. Click **"Share"** button (top-right)
2. Click **"Download"**
3. **File type:** PNG
4. **Quality:** Standard (default is fine)
5. **Size:** Make sure it says **1024 × 1024 px**
6. Click **"Download"**

**Save the file as:** `InboxIQ-Icon-1024.png`

---

## ✅ You're Done!

You should now have a **1024×1024 PNG** file that looks like:
- Blue-purple gradient background
- White envelope in center
- Gold sparkles around it
- Professional and clean

## 🎨 Troubleshooting

**Icon looks blurry:**
- Make sure canvas is exactly 1024×1024 pixels
- Export as PNG (not JPG)
- Check "Standard" quality (not "Low")

**Colors don't match:**
- Double-check hex codes: `#007AFF` and `#5856D6`
- Make sure gradient is linear (not radial)

**Sparkles are too busy:**
- Use fewer sparkles (4-6 instead of 8)
- Make them smaller
- Reduce glow effect

**Envelope is off-center:**
- Use alignment tools (select all → Align → Center)
